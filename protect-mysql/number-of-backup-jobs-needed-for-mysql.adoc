= MySQLに必要なバックアップ ジョブの数
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
必要なバックアップ ジョブの数を左右する要因としては、リソースのサイズ、使用中のボリュームの数、リソースの変更率、サービス レベル アグリーメント（SLA）などがあります。
