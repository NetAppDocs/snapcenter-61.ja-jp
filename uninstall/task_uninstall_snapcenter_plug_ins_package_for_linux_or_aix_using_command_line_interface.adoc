= CLIを使用したPlug-ins Package for LinuxまたはPlug-ins Package for AIXのアンインストール
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenter Plug-ins Package for LinuxとSnapCenter Plug-ins Package for AIXは、コマンドライン インターフェイスを使用してアンインストールできます。

.開始する前に
* スケジュール済みジョブが削除されていることを確認します。
* 実行中のすべてのジョブが完了していることを確認します。


.手順
アンインストールするには、_/custom_location/ NetApp/snapcenter/spl/installation/plugins/uninstall_ を実行します。
