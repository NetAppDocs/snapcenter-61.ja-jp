= SnapCenter softwareのクイック スタート
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
クイック スタート ガイドでは、SnapCenter softwareをインストールして構成するための基本的な手順について説明します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"]SnapCenter Serverのインストールの準備
[role="quick-margin-para"]
SnapCenter Server をインストールするためのすべての要件が満たされていることを確認する必要があります。

[role="quick-margin-list"]
* link:../install/requirements-to-install-snapcenter-server.html["要件"]
* link:../install/register_enable_software_access.html["SnapCenter softwareにアクセスするには登録してください"]
* link:../install/enable_multifactor_authentication.html["多要素認証の有効化"]


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["2"]SnapCenterサーバーをインストールする
[role="quick-margin-para"]
SnapCenter Server は、Windows ホストまたは Linux ホストにインストールできます。  SnapCenter Serverのインストールパッケージを以下のサイトからダウンロードします。 https://mysupport.netapp.com/site/products/all/details/snapcenter/downloads-tab["NetAppサポート サイト"^]インストーラーを実行します。

[role="quick-margin-list"]
* link:../install/task_install_the_snapcenter_server_using_the_install_wizard.html["WindowsにSnapCenterサーバーをインストールする"]
* link:../install/install_snapcenter_server_linux.html["LinuxにSnapCenter Serverをインストールする"]


.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三つ"]SnapCenterサーバーを構成する
[role="quick-margin-para"]
SnapCenter Server をインストールした後、環境に応じて構成する必要があります。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["4"]アプリケーション用のプラグインをインストールする
[role="quick-margin-para"]
使用しているアプリケーションに基づいて、アプリケーション固有のプラグインをインストールするためのすべての要件が満たされていることを確認してから、それぞれのプラグインのインストールを続行します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]アプリケーションを保護する
[role="quick-margin-para"]
SnapCenter Server と必要なプラグインを正常にインストールしたら、アプリケーション バックアップの作成を開始できます。これらのバックアップは、後で必要に応じて復元やクローン作成の目的で利用できます。
