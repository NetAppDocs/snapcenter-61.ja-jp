= SnapCenterによる不要なデータ バックアップの削除の管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterは、ストレージ システム レベルおよびファイルシステム レベルで不要なデータバック アップの削除を管理します。

保持設定に基づいて、プライマリ ストレージまたはセカンダリ ストレージのSnapshotが削除され、PostgreSQLのカタログ内の対応するエントリも削除されます。
