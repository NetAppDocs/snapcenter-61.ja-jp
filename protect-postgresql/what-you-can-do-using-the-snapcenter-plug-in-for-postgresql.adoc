= SnapCenter Plug-in for PostgreSQLの機能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Plug-in for PostgreSQLクラスタをインストールした環境では、SnapCenterを使用してPostgreSQLクラスタとそのリソースをバックアップ、リストア、クローニングできます。これらの処理をサポートするタスクも実行できます。

* クラスタを追加します。
* バックアップを作成します。
* バックアップからリストアします。
* バックアップをクローニングします。
* バックアップ処理のスケジュールを設定します。
* バックアップ、リストア、クローニングの各処理を監視します。
* バックアップ、リストア、クローニングの各処理のレポートを表示します。
