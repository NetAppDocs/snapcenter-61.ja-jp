= サポートされるバックアップのタイプ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
バックアップ タイプでは、作成するバックアップのタイプを指定します。SnapCenterでは、PostgreSQLクラスタに対してSnapshotコピーベースのバックアップ タイプがサポートされます。



== Snapshotコピーベースのバックアップ

Snapshotコピーベースのバックアップでは、NetApp Snapshotテクノロジを利用して、PostgreSQLクラスタが格納されたボリュームのオンラインの読み取り専用のコピーが作成されます。
