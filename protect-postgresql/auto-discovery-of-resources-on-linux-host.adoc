= Linuxホスト上のリソースの自動検出
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
リソースとは、SnapCenterで管理するLinuxホスト上のPostgreSQLクラスタとインスタンスです。SnapCenter Plug-in for PostgreSQLプラグインをインストールすると、そのLinuxホスト上のすべてのインスタンスのPostgreSQLクラスタが自動的に検出されて[Resources]ページに表示されます。
