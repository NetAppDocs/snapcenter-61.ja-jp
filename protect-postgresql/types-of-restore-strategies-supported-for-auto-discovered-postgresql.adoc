= 自動検出されたPostgreSQLでサポートされるリストア戦略のタイプ
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SnapCenterを使用してリストア処理を正常に実行するには、戦略を定義しておく必要があります。

リソースの完全なリストアは、自動検出されたPostgreSQLクラスタに対してサポートされているリストア戦略です。この戦略では、リソースのすべてのボリューム、qtree、およびLUNがリストアされます。
